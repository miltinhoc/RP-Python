# RP-Python
Finds photos uploaded by people to a specific Host. This is an open-source version of RP

### Usage example:
```
rp.py -p m3 -r 200
```

| Argument | Description |
| --- | --- |
| prefix | Sets a String of 2 Characters |
| results | Sets a max limit of search results |
